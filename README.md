
# Logistic Regression From Scratch

## Project Objective
Implement Logistic Regression using only NumPy without using sklearn LogisticRegression.

## Dataset
- Source: Kaggle
- Type: Binary Classification

## Steps Involved
1. Data Loading
2. Data Preprocessing
3. Sigmoid Function
4. Cost Function
5. Gradient Descent
6. Model Training
7. Evaluation (Accuracy, Precision, Recall, F1)

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Results
Custom Logistic Regression model performance is comparable to sklearn model.
